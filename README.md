# FCSE_W25_AdvancedProgramming
Овој репозиториум го користам за решавање на аудиториски и лабораториски вежби по предметот **Напредно програмирање (Advanced Programming)** на **Факултетот за информатички науки и компјутерско инженерство (ФИНКИ)**, во **зимскиот семестар 2025**, **петти семестар**.

Предметот се реализира на програмскиот јазик **Java** и опфаќа теми поврзани со:
- Парадигмата на **генеричко програмирање**  
- Работа со **апстрактни типови на податоци**  
- Креирање на **генерички класи и функции**  
- Користење на **колекции на податоци** (queues, lists, vectors, sets and dictionaries) 
- Вовед во **функционално програмирање** (Java Streams)  
- Примена на **објектно ориентирани дизајн шеми** при моделирање на софтверски решенија
